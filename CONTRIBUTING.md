# Contributing to Portfolio Website

First off, thank you for considering contributing to this portfolio template! It's people like you that make open-source a great community.

## How to Contribute

1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Run tests and linting (`pnpm lint`, `pnpm build`)
5. Commit your changes (`git commit -m 'feat: Add amazing feature'`)
6. Push to the branch (`git push origin feature/amazing-feature`)
7. Open a Pull Request

## Development Setup

See the [README.md](README.md) for detailed instructions on how to set up the project locally.

### Prerequisites

- Node.js 18.17+
- pnpm 8+

## Bug Reports

If you discover a bug in the project, please open an issue and include:

- Steps to reproduce the bug
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

## Feature Requests

If you have an idea for a new feature, please open an issue to discuss it before submitting a PR.
